# Samples

This directory is reserved for synthetic or sanitized sample data only.

Do not store real project documents, private emails, contracts, drawings, tender material, credentials, or personal data here unless the repository owner explicitly approves it.

## Synthetic project fixtures

The `synthetic_project/` directory contains safe, invented fixtures for repeatable tests and demos:

- `synthetic_project/meeting_minutes_001.md`
- `synthetic_project/contract_excerpt_001.md`
- `synthetic_project/project_notes_001.md`

These files are clearly marked `SYNTHETIC SAMPLE - NOT REAL PROJECT DATA`. They must not contain real client names, credentials, emails, phone numbers, or private project identifiers.
